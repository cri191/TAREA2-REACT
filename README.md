# E-commerce - Tarea2-React

## Descripción del Proyecto
Este es un proyecto de E-commerce desarrollado en React  del Módulo 2 trabajo #2. Se agrga conexion con la api y se desarrolla buscador de productos y se rendiriza

## Componentes Creados
* *Header*: Muestra el logotipo y el nombre de la tienda.
* *SearchBar*: Buscador controlado mediante estados para filtrar productos.
* *ProductCard*: Tarjeta reutilizable que recibe propiedades de cada producto.
* *ProductList*: Renderiza la lista completa de productos usando el método map.
* *Button* / *Footer*: Componentes complementarios de la interfaz.

## Tecnologías Usadas
* HTML5 y CSS3
* React
* Vite

## Instrucciones para Ejecutar el Proyecto
1. Clonar el repositorio.
2. Ejecutar npm install en la terminal para instalar las dependencias.
3. Ejecutar npm run dev para iniciar el servidor de desarrollo local.
